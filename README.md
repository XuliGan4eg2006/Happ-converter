# Happ converter
Скрипт для получения vless конфигов из ссылок-подписок.


### Мотивация
Почему просто не пользоваться Happ? Ответ - у него нет открытого исходного кода (есть только [репозиторий-заглушка](https://github.com/Happ-proxy/happ-desktop)), а значит владелец проекта может сделать что угодно не только с своей программой, но и с вашими устройствами (откуда вы знаете, что там нет бэкдора?)

А ещё некоторые vpn провайдеры не дают импортировать подписки в другие приложения (кроме happ)
![app not supported](imgs/appns.png)

## Установка:
Скачайте скрипт:
```bash
git clone 
```

Установите зависимости:
```bash
pip3 install -r requirements.txt
```

## Запуск 
Запустите скрипт и следуйте инструкциям:
```bash
python3 happ_converter.py
```

По окончании работы скрипта вы получите лист vless конфигов, которые уже можете импортировать в любое приложение
